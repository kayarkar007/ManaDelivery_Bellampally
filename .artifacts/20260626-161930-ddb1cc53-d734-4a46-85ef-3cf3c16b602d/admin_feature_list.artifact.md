# Advanced Admin Control Center - Feature Roadmap

Building the ultimate command center for Mana Delivery Bellempally.

## 1. Real-time Logistics Dashboard
- [ ] **Live Order Stream**: Instant view of new, ongoing, and delayed orders.
- [ ] **Rider Map View**: (Conceptual) Track active riders in the area.
- [ ] **Dynamic Revenue Ticker**: Real-time calculated platform earnings (commission based).

## 2. Advanced Vendor Management
- [ ] **Approval Workflow**: Review vendor documents and one-click approve/reject.
- [ ] **Commission Controls**: Set custom commission rates per vendor.
- [ ] **Category Mapping**: Assign vendors to specific hyper-local categories (e.g., "Station Area Special").

## 3. Rider Operations
- [ ] **KYC Verification**: Dedicated screen to verify Rider DL, RC, and ID.
- [ ] **Cash-on-Hand Tracking**: Monitor how much COD money each rider is holding.
- [ ] **Performance Stats**: View successful vs. cancelled deliveries.

## 4. Hyper-Local System Controls
- [ ] **Service "Kill Switch"**: Instantly disable orders for specific areas or the whole town during heavy rain/emergency.
- [ ] **Dynamic Delivery Fees**: Adjust base fee based on distance or time.
- [ ] **Promotions Engine**: Create and manage coupons and banners.

## 5. Automated Communications
- [ ] **Notification Center**: Send manual broadcasts to all Customers, Riders, or Vendors.
- [ ] **Order Alerts**: Automatic push notifications on status changes (Integrated with FCM).
- [ ] **Support Tickets**: Basic in-app resolution center.
